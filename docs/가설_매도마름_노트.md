# 가설 노트 — 극단적 매도 마름 = 발사 신호?

**작성일:** 2026-05-25 (KST)
**상태:** 가설 / 1차 검증 완료 / 단독 트리거 불가 / 재검증 대기

## 가설
원본 알파 신호 직전에 매도 소진(매도액 0, 체결강도 9999 마름)이 선행한다.
세력이 개미 투매를 받아내 매도가 마르는 순간 = 발사 임박.

## 1차 검증 (5/25 알파 3건 CPOOL/SOON/ERA)
- CPOOL 08:50: 직전30분 마름 10회 / 매도만 4회
- SOON 13:05: 마름 8회 / 매도만 19회, 12:50~13:04 매도만연속 → 13:06 마름전환
- ERA 14:43: 마름 9회 / 매도만 7회, 14:28~37 매도만 → 14:42 마름전환

## 결론 — 단독 트리거 불가
매도 마름이 신호 직전에만 아니라 30분 내내 들쭉날쭉 나옴.
이유: 거래 얇음(백만원 단위). 매도 0이면 즉시 강도 9999 → 마름이 노이즈일 수 있음.
30분에 마름 10회면 진짜 발사 직전 구분 불가 → 하드 트리거 불가.

## 정제 여지
단순 마름이 아니라 "매도만 연속(개미 투매) → 마름 전환(세력 흡수)" 시퀀스가 더 의미 있을 수 있음.
SOON/ERA에서 관찰됨. CPOOL은 불명확.

## 보류 이유
체결강도 데이터가 5/25부터만 존재 (수집기 5/24~). 과거 신호 소급 불가. 표본 3건뿐.

## 재검증 조건
1. 체결강도 며칠 누적
2. 신호 시점 vs 비신호 시점의 "매도만→마름 전환" 빈도 비교
3. 전환 시퀀스 정의 확정 (몇 틱 연속?)
4. 거래량 필터 결합 (얇은 거래 노이즈 제거)

## 현재 처리
알람에 매도상태 표시만 (트리거 아님). 일일 성적표로 사후 관찰.
