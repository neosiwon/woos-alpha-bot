## 0. 한 줄 결론
"매수액 누적 = 매집"은 거래량 함정으로 기각. **1틱(1분) 순매수 스파이크**가 세력 매집을 가르는 핵심 지표 — 5/25 알파 3건이 전종목 상위 0.4~2.0%(상위 5위 내), 노이즈는 9%대로 분리됨. 체결강도 150 트리거는 알파 신호 시점을 100% 막던 것으로 확인되어 제거. 신호 = 매집 스파이크 상위(동적 %) + 박스 수축.

## 1. 출발점 — 사용자의 의심
"매수액 값이 매집됐다로 확신할 수 있는가?" → 확신 불가. 검증 결과:
- **당일 누적 순매수(cumNet): 변별력 없음.** CPOOL -24.9억, ERA -12.5억 (음수인데 알파). 누적 상위 = 그냥 거래량 큰 종목(BTC/SOON/USDT 줄세우기).
- 매수/매도 비율, 스파이크 배율, 스파이크 집중도: 모두 알파 3건이 제각각 → 기각.
- **단일 틱 최대 순매수 스파이크: 알파 3건 모두 상위 4% 이내** → 유일하게 변별.

## 2. 검증 여정 (전부 데이터 기각/확정)

### 2-1. 매집 지표 = 순매수 스파이크 (누적 아님)
259종목 비교: 알파 3건이 5분 스파이크 상위 2% 이내. 누적/비율/배율은 변별 실패.

### 2-2. 메이저 제외 = 시총 기준
CoinGecko 시총: 알파 최대 NEAR 4.4조 vs 메이저 최소 ADA 12.5조 (4.4~12.5조 빈 갭).
→ ADA 이상 시총 자동 제외 + XLM(7.1조, 노이즈) 수동 추가. ONDO/SUI/NEAR(ADA 미만)는 후보 유지.
MAJORS = USDT USDC DAI BTC ETH XRP SOL DOGE ADA BNB TRX XLM (12개)

### 2-3. 수축은 매집과 결합해야 의미 있음
- 매집(스파이크 상위)+수축 그룹: +5% 상승 47%
- 매집 안 됨(스파이크 하위)+수축 그룹(15/15 전부 수축): +5% 상승 7%
→ 7배 차이. 수축 단독은 변별력 없으나(명세서 §0 맞음), 매집과 결합 시 강력. 순서 중요: 매집 먼저 거른 뒤 수축.

### 2-4. 체결강도 150 트리거 = 알파 100% 차단 (제거)
5/25 알파 3건 신호 시점 ±5분: 노이즈필터 통과 0건. 체결강도가 0(매도만) 또는 9999(매도마름) 극단.
신호 시점은 거래가 죽은 순간(매집 끝, 발사 직전 수축)이라 체결강도 150 근처가 전혀 아님.
→ 150 게이트 제거. signals_log 실측에서도 옛 트리거 적중 2/20(10%)로 입증.

### 2-5. 매도 소진(0→9999) = 발사 신호 가설 (표시만, 검증중)
알파 3건 신호 직전 공통: 매도만(강도0) 연속 → 매도마름(9999) 전환.
단 변별력 압도적이지 않음(극단값 흔함) → 하드 트리거 아닌 알림 라벨로(추천안).

### 2-6. 약세장/시간대 가설 기각
- **약세장 무관:** 5/22(약세장) 원본 알파 TRAC +14.4%, META +28.6% 도달. 약세장에도 알파 통함.
- **시간대 무관:** 원본 신호가 오전~밤(NEAR 21:15, ONDO 17:15) 분산. 특정 시간대 아님.
→ 신호 노이즈는 시장/시간 탓이 아니라 **트리거 방식** 문제로 확정.

### 2-7. 노이즈의 정체 — 5분 윈도우가 주범 (1틱 역산)
신호 16개 중 진짜 6 vs 노이즈 11 비교:
- 윈도우별 분리도(진짜 평균순위 vs 노이즈 평균순위): **1틱 1.5% vs 9.2% (분리 7.8%p, 최고)**, 5분 1.6% vs 7.0%(5.4%p, 최저)
- 5분은 누적이라 죽은코인도 5분 모으면 커져서 상위로 올라옴(노이즈 부풀림).
- 5/25 알파 3개(CPOOL/SOON/ERA) 1틱 역산: 상위 #5/#1/#4 → **상위 5개 = 알파 3 다포함, 노이즈 0.**
- 5분은 상위 7개 필요 + SUPER(노이즈) 끼임.

## 3. 최종 트리거 로직 (1811caf)
```
[제외] MAJORS (시총 ADA 이상 + XLM)
[1단계 매집] 1틱(1분) 순매수 스파이크 → 동적 상위 TOP_PCT% (MIN/MAX 보정)
[2단계 수축] 박스폭 ≤ BOX_PCT_MAX
[신호] 매집+수축 통과 = 신호 (체결강도 게이트 없음)
[표시] 매도소진 라벨(DRY/SELL_ONLY)·국면(강세/약세)
```
5/25 적용 결과: 16개 → 5개, 알파 3/3 포착, 노이즈 11개 전부 제외.
