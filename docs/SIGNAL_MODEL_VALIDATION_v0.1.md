## 2. 봇 식별 기준 (검증 완료)

## 3. 자전거래 가설 (핵심 발견)

### 모순
세력 봇이 산 것보다 더 많이 팔았다면(순매도), 그 파는 물량은 어디서 나왔나?

### 답: 자전거래 (wash trade)
세력이 계정 2개 이상으로 **계정A 매도 ↔ 계정B 매수**를 같은 봇 수량으로 돌린다. 그래서 순매도(-)여도 물량을 빼앗기지 않고(자기들끼리 순환), 그 와중에 개미 물량을 흡수한다.

### WAL 64682.0071봇 검증 정황 (1회 ≈ 555만원, 32건)
1. 같은 수량이 매수·매도 양쪽에 (소수점 4자리 일치)
2. 6~7초 간격 짝 체결 (매수 2건 / 매도 2건씩)
3. 체결ID 연속 (시간상 매우 가까움 = 봇 자동매매)
4. 단가 일정 (550~585만원)
5. 매집 구간 가격 상승 (85 → 90)

### 한계
거래소가 계정 정보를 주지 않아 "두 계정"을 100% 단정할 수는 없다. 정황이지 증거는 아니나, 정황이 강하다.

### 의미
"순매도인데 왜 매집/발사냐"는 며칠 묵은 모순이 풀린다 → 자전 + 흡수라 순매도가 분배가 아니다. **매집/분배는 단순 순매수 부호로 가를 수 없다.**

## 4. 매집 / 분배 정의 (검증 완료)

**판정 키: 같은 큰 수량 봇이 BID·ASK 양쪽 다 도는가**

| 유형 | 봇 매수/매도 비율 | 패턴 | 사례 |
|---|---|---|---|
| 흡수 매집 | 0.4 ~ 0.8 (매도봇 ≥ 매수봇) | 누르며 흡수, 매물대 有, 박제/하락 | WAL 0.59, FF 0.44, AERGO 0.82 |
| 적극 매집 | 1.2 ~ 3.3 (매수봇 >> 매도봇) | 사올림, 매물대 無, 상승 | H 1.4~3.3 |
| 분배 | 매도봇만 활발 (매수봇 거의 없음) | 일방 매도 + 가격 하락 이탈 | WLD 29368봇 0.29 |

- **단순 순매도(-) ≠ 분배**: WAL/FF는 매도 우위(0.4~0.6)인데 발사함. 매수봇이 계속 돌아 흡수하기 때문.
- 매집 = 양방향 봇 활성 / 분배 = ASK만.

## 5. 발사 2변수 모델 (핵심 프레임)

발사를 **여부**와 **크기**로 분리한다.

### 변수 1 — 발사 여부/근거: 유통량 대비 세력 장악
세력이 거래 가능한 물량을 충분히 장악하면 매물이 부족해져 발사한다.

- H: 빗썸 내부 유통량 1,644만 H (전체 28.3억의 0.058%뿐) × 560원 ≈ 92억 ≈ 봇 매집 90.8억 → 거래 가능 물량 거의 100% 장악 → 발사.
- 측정: 빗썸은 내부 유통량 공시 → 직접 / 업비트는 공시 안 함 → **매집/총거래대금 비율**로 근사.

| 종목 | 매집/거래대금 비율 | 발사율 |
|---|---|---|
| H | 4.7% | +109% |
| WLD | 3.5% | +18% |
| FF | 0.7% | +19% |
| AERGO | 0.5% | +27% |
| WAL | 0.3% | +28% |
| HOME | 0.1% | +17% |
| GNO | 0.0% | +240% (봇 없음, 예외) |

### 변수 2 — 발사 크기/MFE: 위쪽 매물대 두께
발사 전 가격 위쪽에 쌓인 거래량(저항)이 얇을수록 멀리 간다. (60일 일봉 가격대별 거래대금)

| 종목 | 위쪽 매물대 | 바로 위(+50% 이내) | 발사 MFE |
|---|---|---|---|
| **H** | **10.0%** | **2.2%** | **+109%** (위 텅 빔 → 폭발) |
| FF | 34.6% | 34.6% | +19% |
| WLD | 63.1% | 46.9% | +18% (위 두꺼워 막힘) |
| WAL | 100%* | 36.6% | +28% |
| AERGO | 100%* | 99.1% | +27% |
| GNO | 99.3% | 58.8% | +240% (봇 없음, 예외) |

\* 일봉 종가 기준이라 거침 (WAL/AERGO는 최근 하락해 종가가 다 발사 전 가격 위로 잡힘). 정밀화하려면 가격대별 고가-저가 분산을 봐야 함. H(10%)·WLD(63%) 양 끝은 명확.

### 핵심
- **개입 규모 단독으로는 발사 크기를 설명 못 함** (WLD 개입 135억인데 +18%만 = 위 매물대 두꺼워서).
- 발사 여부 = 유통량 / 발사 크기 = 매물대. 두 변수는 섞지 않는다.

## 7. 신호 직전 패턴

원본 신호 시점은 "발사 정각"이 아니라 **거래 죽음(정적)**이다. 발사 20~30분 전 **흡수 클라이맥스**(대량 순매도 + 봇 + 신저점) 후 잠깐 잠잠해진 자리에 신호가 온다.

- AERGO 예: 15:29 -3,537만 흡수 클라이맥스 + 저점 68 → 15:44 신호 → 발사.

## 8. 결론 — 신호 모델

```
[후보] 봇 매집(건당40만+ & 30회+) + 봇 양방향 활성(매집, 분배 제외) + 가격 박제/완만 하락
[발사 여부] 유통량 대비 장악 (빗썸 내부유통량 / 업비트 거래대금 비율)
[발사 크기] 위쪽 매물대 두께 (얇을수록 MFE 큼)
[제외] 봇 0종(GNO형) = 봇 없이 터지는 유형, 우리 영역 밖
```
