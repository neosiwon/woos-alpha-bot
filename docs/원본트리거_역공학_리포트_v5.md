# 원본 트리거 역공학 리포트 v5 — 55건 전수 검증

**작성일:** 2026-05-25 (KST)
**대상:** 원본 채널 "차트에 미친사람들" 매수 신호 55건 (5/9~5/25, 17일)
**목적:** 원본 봇 트리거의 정체 규명 + 우리 봇 노이즈 원인 진단
**방법:** 55건 신호를 업비트 캔들(OHLCV)·5/25 체결 데이터로 전수 분석, 대조군(비신호 종목) 비교

---

## 0. 한 줄 결론
OHLCV(가격·거래량·파생지표)로는 "횡보(78%)"라는 무대 조건까지만 보이고, **진짜 방아쇠는 체결/호가의 미세구조(순매수 매집 + 매도 흡수/소진)에 있다.** 우리 봇이 노이즈가 많은 이유는 트리거를 "1틱 순매수 절대값"으로만 봐서 — 흔한 현상을 잡기 때문. 체결+호가 데이터를 며칠 누적해야 진짜 필터를 검증할 수 있다.

---

## 1. 원본은 봇 트리거다 (전제 확정)
- 메시지가 매번 동일 템플릿의 기계적 반복 → 사람 정성판단 아닌 봇 자동발송
- 모든 신호에 "-8% 자율손절" 명시 → 내부정보(확실한 펌핑)면 손절 불필요. 손절을 거는 건 **확률적 신호**라는 뜻
- → 차트/데이터 기반 트리거. 따라서 데이터로 복제·검증 가능.

## 2. 신호 직전은 횡보 (견고, 78%)
종목별 첫 신호 37개 기준:
- 직전 1시간 변동폭 ≤5%: **78%** (중앙 2.3%)
- 1시간±3% AND 2시간±5% 횡보: 78%
- 예외는 LAYER, DKA 2개뿐 (이미 급등 중)
- **단, 수렴(좁아짐)은 아님** — 변동폭이 시간순 3.1→3.3→4.1%로 오히려 약간 넓어짐. "낮게 유지되는 횡보"지 "수축 삼각형" 아님.
→ 횡보는 트리거가 아니라 **매집이 일어나는 무대 조건**. 유효하나 단독으론 종목 선별 불가(조용한 종목은 수백 개).

## 3. 소거된 가설들 (대조군 검증 실패)
여러 OHLCV 지표를 신호 종목 vs 대조군(비신호 종목)으로 비교 → 대부분 변별력 없음:

| 가설 | 결과 | 비고 |
|---|---|---|
| 박스 수축 5% | ❌ 성과·위험 무관 | 박스폭 0.5~29% 다 +5% 도달 100%, 손절 0건. 비수축(8%+)이 오히려 MFE +28%로 최고 |
| 거래량 수축 | ❌ 49% | 절반만 해당 |
| 변동성 수렴(삼각형) | ❌ 6% | 단조 수렴 거의 없음 |
| 돌파/상승중 | ❌ | 직전 횡보라 돌파 아님 |
| 7일 전고점 위치 | ❌ 분산 | 바닥~천장 골고루, 대조군과 비슷 |
| 전고점 대비 위치 | ❌ | 대조군과 무의미 (시장 전체가 비슷) |
| OBV (1h/2h) | ❌ | 대조군과 비슷 |
| OBV (4h) | △ 약함 | 2배 격차 있으나 표본 작고 5/25 재현 실패 |
| MFI | ❌ | 전 구간 변별력 없음 |
| RSI (1시간봉) | ❌ 결과지표 | 신호 시각엔 65로 높으나, 4시간 前엔 50(대조군과 동일) → 발사 결과지 트리거 아님 |
| 상대 거래대금 순위 | ❌ | (이전 검증) 상위로 좁히면 선점 실패 |
| Basel/Supertrend/LuxAlgo | ❌ | Basel 5/25 직접검증: 신호보다 늦거나 엉뚱(VTHO/XEC/OPEN). Supertrend 남발. LuxAlgo=스윙구조(횡보와 반대) |

**핵심:** 신호 종목은 발사 4시간 전까지 대조군과 구분 안 됨(RSI·거래량·위치 다 평범). 발사는 갑자기 시작되고, 미리 예측할 OHLCV 패턴은 없음.

## 4. IN — 트리거가 OHLCV에 없다는 증거
IN은 트리거가 3번 발동한 표본(5/20 87.5 → 5/23 115 → 5/24 133, 상승하며 재신호).
- 같은 코인 3회 신호인데 **OHLCV 공통점이 없음**: RSI 53~71, 거래량 0.5~2.2배, 전고점위치 0~-14% 다 제각각
- 유일 공통: 직전 24h 가격 플러스(상승 중) — 단 5/20은 +0.6%로 애매, 너무 흔한 조건
→ 트리거가 세 번 발동했는데 OHLCV 공통점이 없다면, **트리거는 OHLCV가 아닌 것(체결/호가)에 있다**는 확증. IN은 반례가 아니라 열쇠.

## 5. 신호 유형 두 가지
- **A형 (횡보 매집)**: 조용히 갇혀 매집 → 한 번 발사. CPOOL, ERA 등 다수.
- **B형 (추세 매집)**: 이미 상승 추세인데 구간마다 재매집 → 지속 상승. IN(+60%), TRAC(+24% 후 재신호).
→ 횡보는 A형에만 해당. 두 유형 공통분모는 **순매수 매집(체결)** 하나.

## 6. 추세 지속 (트레이딩 함의)
55건 4시간 vs 추세최대 백테스트:
- 평균 4h최고 +20.8% / 추세최고 +27.5% / 4h 이후 추가 +6.7%p
- 추세지속(+5%p↑ 더 오름): 29% (IN +55%, MBL +28%, TRAC +16%p 등)
→ B형(추세형)은 4시간 단타로 끊으면 손해. TP1 후 트레일링으로 추세 추종 가치. 단 진입 시점 A/B 구분법은 미해결.

## 7. 5/25 체결 데이터 검증 (표본 한계)
체결(매수/매도 분리) 데이터는 5/25만 존재(수집기 5/24~). 알파 3개(CPOOL/SOON/ERA)뿐:
- **순매수 절대강도**: MAJORS 제외 후 SOON 1위, IN 2위, ERA 4위, CPOOL 5위 → 알파가 최상위 (유일하게 변별력 보인 지표)
- **스파이크 질**(연속성/매수압도): SOON은 신호 시점 매도가 매수의 4배(흡수 정황). 노이즈(TRUST 11.75)가 오히려 매수 압도 → "깨끗한 매수"는 알파 아님
- **매도 마름**: 신호 부근에 마름 몰리나, 얇은 거래라 30분 내내 노이즈로 나옴 → 단독 트리거 불가
→ 3개 표본으론 SOON 하나에 결론이 좌우됨. 통계적 신뢰 불가.

## 8. 세력 매집 모델 (사고실험으로 도출)
세력 관점에서 매집 메커니즘:
```
1. 평단 방어 → 가격을 박스에 가둠 (= 횡보, OHLCV로 보임 ✓ 78%)
2. 매수벽으로 매도 흡수 + 개미 이탈 → 매도량이 시간에 걸쳐 감소 (체결/호가 필요)
3. 매도 물리적 고갈(마름) → 살짝 올리면 추격매수 → 발사
```
- 횡보(원인)와 매도 마름(결과)은 별개가 아니라 인과. 같은 과정의 단계.
- 2번(흡수/소진)이 핵심인데 OHLCV엔 안 보임 → **호가 데이터** 필요.

## 9. 우리 봇 노이즈 진단
- 트리거를 "1틱 순매수 양수"(절대값)로만 봄 → 종목당 하루 수백 번 발생하는 흔한 현상
- 동적컷이 매 틱(60초) 상위 N개 → 시간대마다 바뀌며 누적, 알람 과다
- **개선 방향**: 절대값 → "평소 대비 이상 급증" + 스파이크의 질(연속성/흡수) + 호가(매도벽 소진). 미리 거르는 OHLCV 조건 추가는 무의미(발사 전엔 다 평범).
